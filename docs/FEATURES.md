# Funktioner

Status baseras på aktuell kod.

- Authentication — **Implementerad**: e-post/lösenord, registrering, session och utloggning.
- Podcasts — **Implementerad**: skapa, välja, uppdatera och ägarradering.
- Members and permissions — **Implementerad**: inbjudan via e-post, roller och rollstyrd UI/RLS.
- Dashboard — **Implementerad**: avsnitt, deadlines, statistik, filer och Realtime-aktivitet.
- Episodes — **Implementerad**: skapa, lista, söka, filtrera och full detaljredigering.
- Script — **Implementerad**: editor, statistik, radnummer, monospace och autosparning.
- Segments — **Implementerad**: skapa, redigera, radera och ordna med drag-and-drop.
- Notes — **Implementerad**.
- Checklist — **Implementerad**: produktion och publicering.
- Tasks — **Delvis implementerad**: databastabell/dashboardstöd finns; ingen komplett separat uppgiftssida.
- Material library — **Implementerad**: uppladdning, externa länkar, kategorier, sökning, sortering, preview och borttagning.
- Studio — **Implementerad**: projekt, spår, clips, markörer, närvaro, versioner och aktivitet.
- Recording — **Implementerad**: MediaRecorder, paus, mätare, livevågform, chunk/recovery, Storage-uppladdning och retry.
- Editing — **Implementerad**: markering, multi-select, flytt, trim, delning, delete, copy/cut/paste, duplicate/repeat, gain, fades, mute, undo/redo.
- Publishing — **Implementerad**: metadata, länkar, artwork, status, RSS, checklista och historik.
- Notifications — **Implementerad**: skapande, lässtatus och Realtime.
- Search — **Implementerad**: global och sidspecifik sökning.
- Profile — **Implementerad**: profil, avatar och lösenord.
- Podcast settings — **Implementerad**: podddata, bild, medlemmar, roller, filer och radering.
